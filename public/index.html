Webhook Endpoint

Configure your ElevenLabs webhook to send POST requests to:

https://your-site-name.netlify.app/.netlify/functions/webhook-handler

How it works

  • Receives webhook from ElevenLabs with transcript data
  • Uses OpenAI GPT-4o-mini API to generate a human-readable summary
  • Sends both the summary and full transcript via email to info@beethovenathome.com
  • Uses Mandrill SMTP for reliable email delivery

Setup Instructions

  1. Set up environment variables in your Netlify dashboard:
    • Secrets:
      • OPENAI_API_KEY - Your OpenAI API key
      • MANDRILL_API_KEY - Your Mandrill API key
    • Configuration:
      • OPENAI_MODEL - AI model to use (default: gpt-4o-mini)
      • MANDRILL_USER - Sender email address
      • MANDRILL_HOST - SMTP host (default: smtp.mandrillapp.com)
      • TARGET_EMAIL - Destination email address
  2. Update the webhook URL above with your actual Netlify site name
  3. Configure ElevenLabs to send webhooks to your endpoint
